How-to Guide
Step-by-step instructions for setting up and running your shop
🚀 Getting Started — First Time Setup
Follow these steps once when you first set up the shop
1
Start the server

Double-click run.bat in your project folder.
The server starts at http://localhost:5000

2
Import your products

Come to Admin → Import Products (this panel).
Click "Browse" and select your products.json file.
Choose Upsert mode and click Import Now.

3
Set up payments (Razorpay)

Sign up at razorpay.com (free). Go to Settings → API Keys.
Set environment variables before running: export RAZORPAY_KEY_ID="rzp_live_..."
Supports GPay, Paytm, UPI, Credit/Debit cards automatically.

4
Visit your shop

Open http://localhost:5000 — all products will load automatically.

📦 How to Add or Update Products
Three ways to manage your product catalogue
A
Upload products.json (easiest)

Go to Import Products tab above. Upload your products.json file.
Use Upsert to add new + update existing. Use Replace All to start fresh.
The JSON should be an array: [{"name":"Foxtail Millet","category":"Millets & Grains","price":195,"qty":"1 kg","code":"M-001"}, ...]

B
Use the CLI admin tool

In terminal: cd backend && python3 admin.py add-product
It will ask you for name, category, price, stock etc. interactively.

C
REST API (for developers)

POST /api/admin/products with JSON body. Add header: X-Admin-Key: [your secret]
Fields: name, category, price, code, qty_unit, stock_quantity, is_new_launch

📉 Managing Stock & Out-of-Stock
How the stock system works
1
Go to Manage Products

Each product row has an editable Stock field.

2
Edit the number

Type 0 → shows "Out of Stock" overlay on the product card on the website.
Type 999 → unlimited / untracked (default for most products).
Any other number like 50 → auto-decrements per order.

3
Click "Save Changes"

All edits are saved in one click. The shop updates immediately — no restart needed.

✨ New Launches Carousel
The highlighted carousel at the top of your shop
1
Go to Manage Products

Find the product you want to feature in the New Launches section.

2
Toggle "New Launch"

Click the toggle in the New Launch column. It turns amber when active.
The product immediately appears in the scrollable "Just Landed from Nature" carousel.

3
Toggle it off

Click the toggle again to remove it from the New Launches section.
The product stays in the regular catalogue — it just leaves the carousel.

💳 Going Live with Payments
Accept GPay, Paytm, UPI, cards and more
✅
Razorpay is free to sign up. They charge ~2% per transaction (no monthly fee). Supports all Indian payment methods.
1
Sign up at razorpay.com

Free account. Complete KYC for live payments (takes ~1 day).

2
Get your API keys

Dashboard → Settings → API Keys → Generate Key.
Copy your Key ID and Key Secret.

3
Set environment variables

Before running the server, in your terminal:
export RAZORPAY_KEY_ID="rzp_live_XXXXXXXXXXXXXXXX"
export RAZORPAY_KEY_SECRET="your_secret_here"

4
Deploy with HTTPS

Razorpay requires HTTPS for live payments. Use nginx + certbot (Let's Encrypt) on any VPS.
OR use ngrok for quick testing: ngrok http 5000
